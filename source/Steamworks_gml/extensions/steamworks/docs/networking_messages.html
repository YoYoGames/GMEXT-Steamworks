Networking Messages

Steamworks Interface: ISteamNetworkingMessages

The Networking Messages module of the Steamworks extension is intended to make it easy to port non-connection-oriented code to take advantage of P2P connectivity and Steam Datagram Relay.

While the Networking Sockets module is connection-oriented (like TCP), meaning you need to listen and connect, and then you send messages using a connection handle. Networking Messages, on the other hand, is more like UDP, in that you can just send messages to arbitrary peers at any time. The underlying connections are established implicitly.

See: Steam Networking

General

The following are general functions for working with messages:

  • steam_net_messages_register_callbacks
  • steam_net_messages_unregister_callbacks
  • steam_net_messages_get_last_size
  • steam_net_messages_get_last_steam_id

Sending and receiving messages

The following functions are used for sending and receiving:

  • steam_net_messages_send
  • steam_net_messages_receive_on_channel

Managing sessions with peers

The following functions are used for managing sessions:

  • steam_net_messages_accept_session
  • steam_net_messages_close_session
  • steam_net_messages_close_channel

steam_net_messages_register_callbacks

Steamworks Function: N / A

This function registers the internal Steam Networking Messages callback handler. It should be called once after Steam initialisation.

A call to this function is required to receive async events of type:

  • steam_net_message_on_session_request
  • steam_net_message_on_session_failed

This function operates asynchronously, which means that it does not immediately return the requested result. Instead, upon completion of the task, it will trigger the Steam Async Event.


Syntax:

steam_net_messages_register_callbacks()




Returns:

Boolean


Triggers:

Steam Async Event

This event is triggered when a remote host is sending us a message, and we do not already have a session with them.

Key Type Description
event_type String The string "steam_net_message_on_session_request"
steamid Int64 The user's Steam ID
identity_string String The identity string
identity_type steam_net_identity_type The type of identity
ip String IP address including the port number
is_local_host Boolean Whether this identity is localhost
fake_ip_type steam_net_fake_ip_type The fake IP type


This event is triggered when we fail to establish a connection, or we detect that communications have been disrupted in an unusual way.

Key Type Description
event_type String The string "steam_net_message_on_session_failed"
steamid Int64 The user's Steam ID
state steam_net_connection_state The high level state of the connection
end_reason Real The basic cause of the connection termination or problem (one of ESteamNetConnectionEnd)
flags steam_net_send_flag The send flags
user_data Int64 Arbitrary user data set by the local application code
listen_socket Real Handle to listen socket this was connected on (0 indicates invalid)
remote_pop Real What data center is the remote host in? (0 if we don't know.)
relay_pop Real What relay are we using to communicate with the remote host? (0 if not applicable.)
remote_identity String Who is on the other end? Depending on the connection type and phase of the connection, we might not know
remote_addr String Remote address. Might be all 0's if we don't know it, or if this is N/A. (E.g. Basically everything except direct UDP connection.)
description String Debug description. This includes the connection handle, connection type (and peer information), and the app name. This string is used in various internal logging messages.
debug String Human-readable, but non-localized explanation for connection termination or problem. This is intended for debugging / diagnostic purposes only, not to display to users. It might have some details specific to the issue.

steam_net_messages_send

Steamworks Function: ISteamnetworkingMessages::SendMessageToUser

This function sends a message to a specific Steam user on a given channel.


Syntax:

steam_net_messages_send(steamid64, channel, buffer, size, send_flags)
Argument Type Description
steamid64 Int64 The remote user's SteamID64
channel Real The local channel number
buffer Buffer The buffer containing the data
size Real The number of bytes to send
send_flags steam_net_send_flag The send flags to use




Returns:

Real

-1 on error, positive values indicate a Steam EResult code

steam_net_messages_receive_on_channel

Steamworks Function: ISteamNetworkingMessages::ReceiveMessagesOnChannel

This function receives at most one message from the specified channel. Also records metadata for "last message" helpers.

The function returns one of the following:

  • -1 on error (buffer not found or couldn't write to buffer)
  • greater than or equal to 0 - The number of bytes copied


Syntax:

steam_net_messages_receive_on_channel(channel, buffer, max_size)
Argument Type Description
channel Real The channel number
buffer Buffer The buffer to write the data into
max_size Real The maximum allowed number of bytes to write




Returns:

Real
